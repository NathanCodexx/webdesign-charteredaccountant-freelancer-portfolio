## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser — no build step required

Alternatively, serve locally with any static server:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`

## Customisation

- **Personal info** — Edit text in `index.html` (name, contact details, experience, testimonials)
- **Colours** — Adjust CSS variables in `:root` at the top of `css/style.css`
- **Firm profile download** — Add your PDF and update the `#downloadCv` link in `index.html` and `js/main.js`
- **Images** — Replace files in `images/` and update the corresponding `src` paths in `index.html`
- **Google Maps** — Update the iframe coordinates and `maps.app.goo.gl` link in the Contact section
- **Social links** — Update `href` values in the footer social icons
